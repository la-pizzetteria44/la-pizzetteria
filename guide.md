# Guide d'utilisation du panneau d'administration – La Pizzetteria

## 🔐 Accès à l'interface d'administration

Rendez-vous sur :  
👉 https://la-pizzetteria.fr/gestion

Connectez-vous avec vos identifiants.

💡 Pour des raisons de sécurité, pensez à vous déconnecter après utilisation :

- cliquez sur la photo de profil en haut à droite
- puis sur **"Logout"**

---

## 🧭 Présentation de l'interface

### 🔝 Barre supérieure (menu principal)

- **Contents** : gestion de vos produits (pizzas, boissons, etc.)
- **Media** : bibliothèque d’images
- **Quick add** : ajout rapide d’un produit
- **Profil (image)** : accès à la déconnexion

---

### 📂 Colonne de gauche (Collections)

- Accès à toutes les catégories (Pizzas, Boissons, etc.)
- Cliquez sur une catégorie pour afficher les éléments

---

### 📝 Zone centrale

- En haut : nom de la catégorie sélectionnée
- Bouton **"New ..."** pour ajouter un nouvel élément  
  (ex : *New Pizzas*)

👉 Pour modifier un produit :

- cliquez dessus dans la liste
- modifiez les champs souhaités

---

## ✅ Publier une modification

Après modification :

1. Cliquez sur **"Publish"**
2. Puis sur **"Publish now"**

👉 Les changements seront visibles sur le site.
